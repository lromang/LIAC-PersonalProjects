Análisis exploratorio de datos: R

Una vez que las bases están limpias y listas para usarse, podemos hacer un poco de análisis exploratorio de datos para darnos una idea de como se comportan los usuarios. Una pregunta interesante podría ser si varía el número de usos dependiendo de si el usuario es hombre o mujer. 

usu_usu <- ggplot(data = usuarios,
                                aes(x = usuario,
                                       y = usos,
                                       color =sqrt(usos) )) +
                    geom_point(size = .7, alpha = .8) +
                    facet_wrap(~sexo) +
                    theme +
                    scale_colour_gradient(low = color2, high= color1)
